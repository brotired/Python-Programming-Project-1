# To Do List Python Program

## Description
This is a simple To Do List program built using Python.  
The program allows users to add tasks to a list and view their tasks.

This project was created to practice Python lists, including:
- Adding items to a list using `append()`
- Displaying items using a `for` loop
- Using user input and menu options

## Features
- Add new tasks
- View all saved tasks
- Simple menu system
- Keeps running until the user chooses to exit

## How It Works
1. The program starts with an empty task list.
2. Users can choose to add a task.
3. New tasks are stored using a Python list.
4. Users can view all tasks stored in the list.
5. The program ends when the user selects the exit option.

## How to Run

1. Make sure Python is installed.
2. Open the project folder in VS Code.
3. Run the following command in the terminal:

```bash
python ToDoList.py
